CLI

question 1: What is CLI?
Answer:

Command line interface, which processes commands to a computer program in form of line of text.

question 2: What is the advantage/disadvantage of CLI?
Answer:

    Advantage:
  • Save time to handle (like create many folders, files,..)
  • Automation (When deal with complex or huge tasks)
  • Easier to remote work
  • Not all software has an interface to use, especially for developers (System, Network,..)
      • Disadvantage:
  • Confusing (Learning and remembering commands sometime is hard, some command is not easy to understand)

    question 29: List all values of position property? Which one is default value?
    Answer:

    There are 5 positions property:

  • static: the elements is positioned according to normal flow of the document -> this is default value
  • relative: positioned according document, but offset relative to itself.
  • absolute: position relative to closest positioned ancestor.
  • fixed: relative to containing block
  • sticky: offset relative to nearest scrolling ancestor
